## 🌟 About The Project

> **Delicious Bites** is a beautifully crafted, fully responsive bakery website designed to make your mouth water from the first click. Built with clean HTML5, expressive CSS3, and the power of Bootstrap 5 — this project brings a warm, inviting bakery experience straight to the browser.

Whether you're browsing on a phone or desktop, Delicious Bites serves up a seamless experience with smooth UI components, gorgeous dish displays, and interactive offer popups.

---

## ✨ Features

| 🎯 Feature | 📋 Description |
|---|---|
| 📱 **Responsive Navbar** | Collapsible menu — perfect for mobile & desktop |
| 🦸 **Hero Section** | Eye-catching welcome banner with call-to-action |
| 🍽️ **Featured Dishes** | 3 highlighted menu items with images & order buttons |
| 🏷️ **Special Offers** | Bootstrap modal popups with detailed deal info |
| 🔗 **Footer** | About info + quick navigation links |
| 🎨 **Custom CSS** | Hand-crafted styles layered on top of Bootstrap |

## 🚀 Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/itxashancode/Delicious-Bites.git

# 2. Navigate to project folder
cd Delicious-Bites

# 3. Open in browser
open index.html
# or just double-click index.html 🎉
```

> No dependencies, no installs — pure HTML/CSS/Bootstrap magic!
